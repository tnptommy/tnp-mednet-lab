## Project Purpose

The purpose of this project is to design and document a realistic enterprise healthcare network supporting secure, resilient, and monitored connectivity across multiple healthcare sites in Ho Chi Minh City.

## Scope

In scope:

- Data centre network design
- MPLS L3VPN provider core
- Metro hospital branch
- Remote community clinic branch
- Sophos and pfSense firewall design
- Windows Server 2022 infrastructure
- Ubuntu 24.04 Docker monitoring stack
- Wazuh and Splunk monitoring
- BA and technical documentation
- Testing and operational handover

Out of scope:

- Real patient data
- Production healthcare system deployment
- Real telco service contracts
- Real cloud billing or production Azure workloads
