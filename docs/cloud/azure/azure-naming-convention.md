# Azure Naming Convention

All Azure resources should follow the TNPH naming standard.

## Format

TNPH-RESOURCE-WORKLOAD-ENVIRONMENT-REGION-NUMBER

## Region Code

- Australia East: AUE

## Examples

- TNPH-RG-MEDNET-LAB-AUE-001
- TNPH-VNET-HUB-LAB-AUE-001
- TNPH-SNET-MGMT-LAB-AUE-001
- TNPH-SNET-SHARED-LAB-AUE-001
- TNPH-SNET-SECOPS-LAB-AUE-001
- TNPH-SNET-PEP-LAB-AUE-001
- TNPH-NSG-MGMT-LAB-AUE-001
- TNPH-NSG-SHARED-LAB-AUE-001
- TNPH-NSG-SECOPS-LAB-AUE-001
- TNPH-RT-HUB-LAB-AUE-001
- TNPH-LAW-MEDNET-LAB-AUE-001

## Storage Account Exception

Storage account names must be lowercase and globally unique.

Primary option:

- tnphstmednetlab001

Backup options:

- tnphstmednetlab002
- tnphstmednet2026
- tnphsttnpmednet01
